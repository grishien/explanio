Chrome extension to explain highlighted text 1

1. Select phrase. Press Ctrl + Alt + Num-
2. Select context (wider text). Press Ctrl + Alt + Num+
3. Look to the right.

F9 - show / hide right sidebar

Before usage - make sure you installed ollama + pulled model "Deepseek 1 8b" OR you have openrouter.ai API key.
Set it up in settings panel.
